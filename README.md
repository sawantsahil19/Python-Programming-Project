# Python-Programming-Project
### Web Scraping, EDA &amp; Data visualization of ‘bikewale.com’ 

Conducted Exploratory Data Analysis and performed univariate and bivariate analysis on motorcycle dataset using Python libraries (Pandas, Matplotlib, Seaborn) to extract insights on engine capacity, mileage, power, and pricing.

Built visual dashboards to identify brand performance, price-power correlation, and customer rating trends.

Derived actionable insights highlighting market segmentation, market performance, and brand positioning to support business decisions.
